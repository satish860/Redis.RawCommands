Redis.RawCommands

Minimalist Framework Written in C# for redis .

Getting Started

Raw Commands library consists of classes which helps you to connect to redis and allow you to execute the commands as you do in the Redis CLI.

Nuget Package

Will be availble in 27 of this month with the beta version

Connecting to Redis

 using Redis.RawCommands
 IRedisClient client=new RedisClient();

It connects to the redis with the default as Localhost and the port as 6379 .
